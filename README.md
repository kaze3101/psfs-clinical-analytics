# PSFSを用いた患者教育がリハビリテーション改善度に与える影響の分析

## プロジェクト概要
本プロジェクトは、回復期（または急性期・維持期等）リハビリテーションにおいて、**PSFS（Patient-Specific Functional Scale：患者特異的機能尺度）**を用いた患者教育が、身体機能の改善効率にどのような影響を与えるかを統計的に検証したものです。

単なる機能訓練に留まらず、患者自身が設定した目標に対する「気づき」を促す教育的介入の有効性を、データサイエンスの手法を用いて可視化しました。

## 使用技術
- **Language:** Python 3.9
- **Libraries:** Pandas (データ処理), Matplotlib / Seaborn (可視化), SciPy (統計検定)
- **Environment:** Jupyter Notebook

## 分析手法
1. **データクリーニング:**
   - 同一患者で「初期評価」と「最終評価」の2点データが存在する症例のみを抽出。
   - 基準日（2024年5月1日）を境に、介入前後の2グループに分類。
2. **評価指標の算出:**
   - 各患者のPSFS改善量（最終スコア - 初期スコア）を算出。
3. **統計的検証:**
   - 記述統計量の算出。
   - ウェルチのt検定（Welch's t-test）による有意差の検討。
   - 効果量（Cohen's d）の算出。

## 分析結果
介入前後のグループ比較において、以下の結果が得られました。

| 指標 | 数値 |
| :--- | :--- |
| **t値 (t-statistic)** | 1.049 |
| **p値 (p-value)** | 0.296 |
| **効果量 (Cohen's d)** | 0.182 |



### 考察と今後の課題
統計的な有意差（p < 0.05）および大きな効果量は認められませんでした。この結果から、以下の要因が推察されます。
1. **介入の標準化:** 教育の導入初期であったため、スタッフ間でのフィードバック内容や頻度にバラつきがあった可能性。
2. **サンプルサイズ:** 有意な差を検出するための症例数が不足していた可能性。
3. **天井効果:** 対象者の初期能力によって、改善の余地に差があった可能性。

**結論として、** 本分析を通じて、単に評価指標を導入するだけでなく、介入の「質」を担保するための標準化が必要であることが浮き彫りになりました。今後はこの分析結果を元に、教育プロトコルの再構築と、より大規模なデータでの再検証を計画しています。
